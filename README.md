# Music Recommendation System Based on Facial Emotion Recognition

## Overview

This project recommends music based on a user's facial emotions detected through a webcam. Using computer vision and machine learning techniques, the system identifies emotions such as Happy, Sad, Angry, Neutral, and Surprised, then recommends songs that match the user's mood and preferences.

## Business Objective

The goal of this project is to improve user experience by providing personalized music recommendations based on real-time emotion detection and user preferences.

## Features

- Real-time facial emotion detection
- Personalized music recommendations
- Language and singer-based filtering
- Interactive web interface
- Emotion-driven playlist generation

## Technologies Used

- Python
- OpenCV
- Pandas
- NumPy

## Workflow

1. Capture user image through webcam
2. Detect face and extract facial features
3. Identify user emotion
4. Collect preferred singer and language
5. Generate personalized music recommendations
6. Redirect user to recommended songs

## Data Analysis Components

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Emotion Distribution Analysis

## Key Metrics

- Emotion Detection Accuracy
- Recommendation Success Rate
- User Engagement Rate
- Most Common Emotions
- Popular Languages and Singers

## Key Insights

- Happy and Neutral emotions were most frequently detected.
- Recommendation quality depends on emotion detection accuracy.
- User preferences significantly influence song recommendations.
- Personalized recommendations improve user engagement.
